CSS Tables

border-collapse: collapse;: Merges the borders of table cells to create a single border.
thead Styling: Adds a background color to the table header.
th, td Styling:Sets the border, padding, and text alignment for table headers and cells. Applies a background color and text color for the header cells.
Row Striping: Uses nth-child(even) to apply a background color to even rows, creating a striped effect.
Hover Effect: Changes the background color of a row when the user hovers over it.
